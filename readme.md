## Instalación

	npm install

## Configurar antes la conexión con la BBDD en app.js

	host: 'localhost',
	user: 'root',
	password : 'root',
	port : 3306,
	database:'demo_clientes'

Crear BBDD 'demo_clientes' y ejecutar en ella el script clientes.sql

## Ejecución

	npm start

## Navegador: Introducir la siguiente url:

	http://localhost:8080/
